# SIJAWA API

SIJAWA API adalah backend service untuk **SIJAWA (Sistem Jadwal Mahasiswa)**.  
Service ini menyediakan REST API untuk mengelola jadwal perkuliahan, tugas kuliah, dan data akademik lainnya.
